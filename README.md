# 简介使
邮箱代理是一个期末大作业项目。邮箱代理能够实现QQ、网易邮箱的发送和接收，并且能够支持下载邮件到本地。同时拥有相对完善的用户交互界面。
# 如何使用
直接下载python代码，将所有的库全部导入，即可运行。
进入主窗口选择自己的邮箱类型，填写邮箱号，填写授权码即可进行邮件的发送和下载。
